A terminal-based implementation of the classic Snake game, written in C and rendered using the ncurses library. In addition to standard player-controlled gameplay, the project includes an AI mode that autonomously navigates the snake toward food using a greedy, Manhattan-distance-based pathfinding algorithm. To build, run gcc main.c -o snake -lncurses; to play, run ./snake and use the arrow keys to move, A to toggle AI mode, R to restart, and Q to quit.
